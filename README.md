# HTML & CSS Mini Practice Projects

This folder contains three beginner-friendly HTML and CSS practice projects. These projects help in understanding layouts, styling, Flexbox, buttons, navigation, hover effects, and basic UI design.

---

## 1. Profile Card

A simple profile card displaying a user's photo, name, short bio, and a call-to-action button.

### Features
- Circular profile image
- Name and short bio
- Styled card with shadow
- Rounded corners
- Hover effect on button
- Centered layout using Flexbox

### Skills Practiced
- HTML structure
- CSS styling
- Flexbox
- Box Shadow
- Border Radius
- Hover Effects

---

## 2. Navigation Bar

A clean horizontal navigation bar with menu links and hover effects.

### Features
- Horizontal navigation menu
- Responsive Flexbox layout
- Hover effects on menu items
- Simple and modern design

### Skills Practiced
- Navigation menus
- Flexbox
- CSS Selectors
- Hover Effects
- Spacing and Alignment

---

## 3. Button Collection

A collection of different button styles demonstrating various CSS properties and hover effects.

### Buttons Included
- Primary Button
- Success Button
- Danger Button
- Outline Button
- Ghost Button

### Skills Practiced
- Button Styling
- CSS Transitions
- Borders
- Border Radius
- Hover Effects

---

## Technologies Used

- HTML5
- CSS3

---

## Learning Outcomes

After completing these projects, you will understand:

- Creating reusable UI components
- Structuring web pages with HTML
- Styling elements using CSS
- Using Flexbox for layout
- Applying hover effects and transitions
- Designing clean and responsive interfaces
